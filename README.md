# ignite-reactjs-challenge-01

Project carried out through the Rocketseat Ignite course.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install depedencies.

```bash
yarn install
```

## To start the project

```bash
yarn dev
```
